A planilha funciona como um formulário organizado para coleta de dados pessoais, financeiros e bancários de um titular, provavelmente para uso em declaração de imposto, controle financeiro ou envio de informações a um escritório contábil.

Coleta informações pessoais essenciais, como:

Nome, CPF, data de nascimento
Título de eleitor
Dados de endereço
Telefone, celular e e‑mail
Informação sobre cônjuge e dependentes
Indicação se houve alteração desde a última entrega

Função: Registrar dados cadastrais completos do titular.

Permite lançar os valores que o titular possui em diferentes bancos. Inclui:

Código e nome do banco
Valor atual (saldo ou rendimento)
Upload do arquivo correspondente (ex.: PDF do informe)

Também calcula um TOTAL dos valores informados.
Função: Registrar rendimentos financeiros e anexar comprovantes.

Lista entradas de receita, como:

Data
Categoria (ex.: holerite)
Valor recebido

Função: Controlar mensalmente as entradas financeiras do titular.
